Use [TypeBox](https://github.com/sinclairzx81/typebox) to define your yargs arguments.

## Install

### NPM

```bash
npm install yargs-typebox
```

### Yarn

```bash
yarn add yargs-typebox
```

### PNPM

```bash
pnpm add yargs-typebox
```
